Project Description

This web application was devoloped using Neo4j and Flask that allows users to view, explore, and manage family trees. The platform provides an interactive family tree visualization, with features such as SVG downloads, biography pages for each family member (including contact details, descriptions, and an embedded Google Maps location), and a comments section. Users can request access to other family trees and apply for admin privileges. Admins have full control over the platform, allowing them to edit nodes, update biography pages, manage user requests, create new trees, and monitor activity via a notification center. The frontend, built with HTML, CSS, and JavaScript, ensures a smooth user experience, while Google Maps API integration enhances location-based details. This project highlights my skills in full-stack development, database management, and interactive data visualisation.
